Coconuts in Space

Date: 2017-05-08

Screenshots:

Let me set the scene. The year is 2017. I'm 12 years old. Already a recipe for disaster. Then, I started using Unity. Technically, this is the 2nd game I made with the engine, but in practice it's the one that matters. Despite what you might think if you try it, I had actually made games prior to this point too! I had fundamental knowledge of GameMaker Studio, which amounted to nothing, and besides that, well...

I had some experience programming, though that was mostly in Visual Basic, and moreover it was primarily experienced gained from copy n' pasting code from other people's tutorials, scratching my head at why it didn't work, and then trying another tutorial. Not exactly an empirical science. I'd like to think I've gotten better since then - jury's out on that - but this game serves as a time capsule to a time when I thought that the only way to let the player reload each stage was to have a dedicated script for each stage where the only difference was an increase in scene index. 10/10.

I don't recommend trying this game out, because there's nothing to gain out of it. It's a bad game and it isn't fun. It's here for archival purposes. At least it shows how far I've come.
